# Быстрый старт

## 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

## 2. Настройка .env файла

Создайте файл `.env` со следующим содержимым:

```
BOT_TOKEN=8564606241:AAEfxSP16OO7LiJRDe0nLUjDD3dFGKldAlw
LAWYER_CHAT_ID=ваш_chat_id
```

**Как получить LAWYER_CHAT_ID:**
1. Напишите боту @userinfobot в Telegram
2. Он покажет ваш Chat ID (число)
3. Скопируйте это число в `.env`

## 3. Запуск

### Локально:
```bash
python bot.py
```

### Через Docker:
```bash
docker-compose up -d
```

## 4. Отладка

1. Откройте `bot.py` в Cursor/VS Code
2. Установите breakpoint (кликните слева от номера строки)
3. Нажмите **F5**
4. Бот запустится в режиме отладки

## 5. Тестирование

1. Найдите вашего бота в Telegram
2. Отправьте команду `/start`
3. Проверьте работу всех кнопок и формы заявки

## Важно!

- Токен бота уже указан в примере
- Не забудьте указать `LAWYER_CHAT_ID` для получения заявок
- Бот должен быть запущен постоянно для работы
